Multi line ellipsis Angular directive

This is a reusable angular directive that provide support for multiline ellipsis.

It is based on the isea presented by Mobify

Note: this implementation assumes that the content wraps and uses height to determine overflow situation. In cases where content gets truncated due to its width, this implementation is not good enough.

You can download or fork from GitHub
